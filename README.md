# Atividades Lab III - Professor Bertoti

Este repositório contém os projetos desenvolvidos para a disciplina de Laboratório III no 3º semestre de Banco de Dados.

## 📁 Organização
- **/lab3/projeto1**: API REST Spring Boot (Java) com CRUD completo (GET, POST, PUT, DELETE) e banco H2.
- **/lab3/projeto2**: Evolução do projeto anterior integrando Front-end (HTML/CSS) com o Back-end.

## ☕ Como Executar
1. Clone o repositório.
2. Importe o projeto desejado no IntelliJ como um projeto Maven.
3. Execute a classe principal.
4. Para o Projeto 2, acesse `http://localhost:8080`.
